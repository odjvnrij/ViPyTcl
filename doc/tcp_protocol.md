# Tcl间通讯协议
## -------------- 消息头 16 Bytes --------------
### 4 Bytes 消息起始 固定为 0x54545454
### 4 Bytes 消息长度, 大端序，长度为消息头和消息体的总字节数

### 2 Bytes 消息类型, 大端序
- 0x0100: 心跳包
- 0x0200: 命令请求
- 0x0201: 命令响应
- 0x0300: 文件传输请求
- 0x0301: 文件传输响应
### 1 Byte 大版本号
### 1 Byte 小版本号

### 4 Bytes 预留字节，全为 0

## -------------- 消息体 N Bytes --------------
### N Bytes 消息

## -------------- 0x0200 消息请求消息体 N Bytes --------------
### 3 Bytes timeout，单位 sec
### 1 Bytes 状态码
### N - 4 Bytes 命令

## -------------- 0x0201 消息相应消息体 N Bytes --------------
### 3 Bytes timeout，单位 sec
### 1 Bytes 状态码
### N - 4 Bytes 回复

## -------------- 0x0300 文件传输请求消息体 8 + x Bytes --------------
### 3 Bytes timeout, 单位 sec
### 1 Bytes 状态码
### 4 Bytes 文件内容大小
### x Bytes 文件内容

## -------------- 0x0300 文件传输响应消息体 8 + x Bytes --------------
### 3 Bytes timeout, 单位 sec
### 1 Bytes 状态码
### 32 Bytes 文件MD5校验值
